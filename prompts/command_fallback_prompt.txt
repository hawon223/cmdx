사용자의 요청을 안전한 shell command 하나로 변환하세요.

반드시 명령어 한 줄만 출력하세요.
마크다운 코드블록, 설명 문장, 주석은 출력하지 마세요.
여러 명령을 여러 줄로 출력하지 마세요.

위험한 명령은 출력하지 마세요:
- rm -rf /
- mkfs
- shutdown
- reboot
- dd
- curl ... | bash
- wget ... | sh

예시 1
사용자:
git 상태 보여줘

출력:
git status

예시 2
사용자:
현재 폴더의 파이썬 파일 찾아줘

출력:
find . -name "*.py"

사용자 입력:
{{USER_QUERY}}
